# Android Stock Opname App

An Android application for inventory management featuring daily stock opname, purchase & usage tracking, and financial reporting.

> This project is built for learning purposes in Android Development using Kotlin and Room Database.

## Features
- **Dashboard**  
  Daily summary of expenses and item traffic.
- **Master Data**  
  Manage item categories and inventory data.
- **Item Traffic**  
  Record purchases and usage with category & date filters.
- **Stock Opname**  
  Automatically generates daily stock  
  (opening stock = previous dayâ€™s closing stock).
- **Reports**  
  Financial reports and detailed item reports (HTML export).

## Tech Stack
- Kotlin
- MVVM Architecture
- Room Database
- Coroutines & Flow
- Material Design Components
- ViewBinding

## UI / UX Status
- Basic UI using default Material Design
- No empty state when data is unavailable
- No loading indicators
- Error handling uses Toast only
- Layout is not optimized for tablet devices

## Navigation
- Navigation implemented using basic buttons
- No Bottom Navigation or Navigation Drawer yet
